## Kurulum

### Gereksinimler
- .NET 10.0 SDK
- PostgreSQL veya [Supabase](https://supabase.com) hesabı (ücretsiz tier yeterli)
- [Groq](https://console.groq.com) API anahtarı (ücretsiz)

### 1. Repoyu Klonla
```bash
git clone https://github.com/frknaydn3512/JobAnalyzer.git
cd JobAnalyzer
```

### 2. Ortam Değişkenlerini Ayarla
```bash
cp .env.example .env
```

`.env` dosyasını düzenle:

| Değişken | Açıklama | Zorunlu |
|----------|----------|---------|
| `DEFAULT_CONNECTION` | PostgreSQL bağlantı dizesi | ✅ |
| `GROQ_API_KEY` | Groq API anahtarı (AI analiz) | ✅ |
| `ADZUNA_APP_ID` / `ADZUNA_APP_KEY` | Adzuna scraper | Hayır |
| `JOOBLE_API_KEY` | Jooble scraper | Hayır |
| `IYZICO_API_KEY` / `IYZICO_SECRET_KEY` | Ödeme entegrasyonu | Hayır |
| `IYZICO_BASE_URL` | `https://sandbox-api.iyzipay.com` (test) | Hayır |
| `IYZICO_CALLBACK_URL` | `https://yourdomain.com/Payment/Callback` | Hayır |

### 3. Veritabanı Migration
```bash
dotnet ef migrations add InitialPostgres \
  --project JobAnalyzer.Data \
  --startup-project JobAnalyzer.Web

dotnet ef database update \
  --project JobAnalyzer.Data \
  --startup-project JobAnalyzer.Web
```

### 4. Web Uygulamasını Başlat
```bash
dotnet run --project JobAnalyzer.Web
# → https://localhost:5001
```

### 5. İlk Admin Hesabı
`/Account/SetupAdmin` sayfasını ziyaret ederek ilk admin hesabını oluştur.

### 6. Veri Toplama
```bash
# İnteraktif menü — istediğin scraper'ı seç veya tümünü çalıştır
dotnet run --project JobAnalyzer.Scraper
```

### 7. AI Analizi Başlat
Scraper menüsünden **"Groq AI Analiz"** seçeneğini çalıştır.  
Veya Hangfire dashboard'undan (`/Admin`) otomatik görev tetikle.

## Otomatik Görevler (Hangfire)

| Görev | Zaman | Açıklama |
|-------|-------|----------|
| `daily-scraping` | Her gece **03:00** | Tüm scraper'ları çalıştır + AI skill extraction |

## Güvenlik

| Alan | Uygulama |
|------|----------|
| Auth | ASP.NET Identity + rol tabanlı yetki |
| CSRF | `AntiForgeryToken` tüm POST formlarında zorunlu |
| PDF | Magic bytes kontrolü (`%PDF` imzası) — sahte dosya yüklemesi engeli |
| AI Prompt | Regex sanitizasyon + `temperature: 0` + output uzunluk validasyonu |
| Ödeme | Iyzico callback işlemi `BeginTransactionAsync` ile atomik |
| Brute Force | Account lockout (5 başarısız giriş → 15 dk kilitleme) |
| SQL Injection | EF Core parametreli sorgular |
| Performans | 7 composite DB index (scraping & filtreleme sorguları için) |
